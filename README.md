# FOX
this css framework is grid design.

## Usage
  ### cssファイルの読み込み
    <link rel="stylesheet" href="css/framework.css">
  ### 12分割になっているので、それに合わせて分割数を設定する（下記では4分割）
    <div class="container">
      <div class="row">
        <div class="four columns">four</div>
        <div class="four columns">four</div>
        <div class="four columns">four</div>
      </div>
    </div>
